# Basic Syntax of Programming Languages

__Covered Concepts:__

* Basic Program Structure
* Variables & Types:
    * Value Types
    * Reference Types
* Expressions:
    * Arithmetic Expression
    * Logical Expressions
    * Order Expressions
    * Equality Expressions
* Statements:
    * Iteration Statements
    * Selection Statements
    * Exception Handling
